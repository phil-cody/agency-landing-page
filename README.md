# Agency Landing Page

## Описание
Современный адаптивный лендинг для агенства

## Технологии

 - HTML5 (семантическая разметка)
 - SCSS (модульная архитектура)
 - BEM методология
 - CSS Flexbox
 - CSS Grid Layout
 - Responsive Design (Desktop First)
 - JavaScript: Sidebar menu (переключатель по бургер-кнопке)

## Особенности
 - Полностью адаптивный дизайн (mobile, tablet, desktop)
 - Модульная структура SCSS с компонентами
 - Использование BEM для именования классов
 - Оптимизированная типографика с системой пресетов
 - Использование JavaScript для реализации Sidebar menu (переключатель по бургер-кнопке)

## Компоненты
 - Header
  * Логотип
  * Навигация
  * Кнопка CONTACT
  * Бургер меню + мобильное меню
 - Hero
  * Заголовок
  * Индикатор прокрутки 
 - Services
  * Transform you brand
  * Stand out to the right audience
 - Projects
  * Graphic Design
  * Photography
 - Testimonials
  * Отзывы клиентов (аватар, текст, имя, должность)
 - Image Grid
 - Footer
  * Логотип
  * Навигация
  * Социальные сети

## Breakpoints
 - Mobile – от 375px до 768px
 - Tablet – до 1440px
 - Desktop – 1440px+
